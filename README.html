Ответы на вопросы: 1.HTML формы нужны для передачи информации программам-обработчикам. События blur указывает пользователю на ошибку введения обязательного поля. События: cut, copy, paste нужны для разрешения, либо запрета копировать/изменять информацию. Отправка нужна для проверки формы перед входом на сервер. Например, пользователь вводить логин от своего личного кабинета, а событие не срабатывает, таким образом любой пользователь может посетить ваш личный кабинет. 2.Возможности JavaScript я бы могла использовать при разработке многих функционалов. Например: Онлайн-заказ продуктов. Многие формы могли бы использоваться на этом ресурсе: дата, логин, фио, отправка и т.д. Так же формы можно использовать при создании сайта ЕГЭ и различных подготовок к экзаменам. Хорошей идеей использования форм будет являтся сайт, где можно расчитать выплаты по ипотеке и др. 3.В наше время на каждом втором сайте используются формы. Например: Вконтакте при регистации/авторизации. https://vk.com/
